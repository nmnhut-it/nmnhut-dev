# SFX assets

Drop one file per spell here (mp3, decoded via Web Audio). The app runs fine with
these missing — VFX just plays silently until you add them.

| File | Spell | Suggested sound |
|---|---|---|
| `fireball.mp3`  | Fireball        | whoosh + fire ignite |
| `frost.mp3`     | Frost Spike     | ice crackle / crystalline chime |
| `lightning.mp3` | Chain Lightning | electric zap / thunder crack |
| `shield.mp3`    | Arcane Shield   | magic barrier hum / shimmer |
| `portal.mp3`    | Portal          | whoosh + reverse riser |
| `heal.mp3`      | Healing Aura    | soft chime / sparkle |

## Free, license-clear sources

- **Kenney.nl** (CC0, no attribution needed) — https://kenney.nl/assets?q=audio — "Magic SFX" and "Impact Sounds" packs have ready-made spell sounds.
- **Mixkit** (free license, no attribution) — https://mixkit.co/free-sound-effects/magic/
- **Freesound.org** (mixed CC licenses — check each upload's license, many require attribution) — https://freesound.org/search/?q=magic+spell
- **OpenGameArt.org** (CC0/CC-BY, game-oriented) — https://opengameart.org/art-search?keys=magic+sound

Rename whatever you download to match the filenames above.
