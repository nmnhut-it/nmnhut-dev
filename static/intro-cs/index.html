The idea

A machine sees data, not meaning.

You see a triangle instantly; a camera hands the machine a grid of 0s and 1s. This course is about that gap: choose how to represent a problem as data, then write exact Python rules over it. Use this page to walk the units, build a plan, take checks, and read the code.
