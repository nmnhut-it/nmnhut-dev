Introduction to Computer Science in Python

A high-school course (≈15 years old) with the MIT 6.0001 lectures as the backbone, adapted down. Python from Unit 1; representation-first as the hook.

A machine perceives the world as data — differently from you. You turn a problem into data the machine can hold, then write exact rules over it.
